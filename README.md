# The OPEC Crisis 🛢️

A browser-based AI geopolitical simulation. You play as **Sheikh Mohammed bin Zayed**, President of the UAE, navigating the 2026 OPEC crisis and Iran war across 13 turns.

Every scenario, reaction, and consequence is dynamically generated by **Gemini 2.0 Flash**.

## Play it

👉 **[Launch on GitHub Pages](https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/)**

You'll need a free Google AI API key from [aistudio.google.com](https://aistudio.google.com/app/apikey).

## Deploy in 3 steps

1. Fork or upload this repo to GitHub
2. Go to **Settings → Pages → Source: Deploy from branch → main → / (root)**
3. Visit `https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/`

## How it works

- Pure HTML/JS — no build step, no server, no dependencies to install
- Your API key is entered at runtime and held only in memory (never stored or transmitted anywhere except the Gemini API)
- Uses `gemini-2.0-flash` — a full playthrough costs fractions of a cent
